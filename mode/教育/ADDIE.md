# 教学系统设计的经典模型：ADDIE

1. 分析（analysis）
   1. 分析受众需求
   2. 分析学习者知识，情感，动作，技能，态度应该掌握的程度，以能做什么为界定的条件。
   3. 分析学习者的起点技能和动机特征
   4. 分析可利用的资源，条件和限制
2. 设计（design）
   1. 教学目的转换成表现型结果和单元目标
   2. 确定每个单元的时间
   3. 充实教学单元，确定内容和学习活动
   4. 开发出评价习得的标准
3. 开发（development）
   1. 学习资料
   2. 试用
   3. 教师材料和辅助材料
4. 实施（implementation）
5. 评价（evaluation）
