# 好奇心缺口

“好奇心缺口”指的是当我们觉得自己的知识出现缺口时，好奇心就产生了。有缺口就会有痛苦，当你想知道一件事却不知道的时候，就好像身上很痒，总想伸手去挠一样。要解除这种痛苦，就得填满缺口。这个说法最早来自美国卡内基梅隆大学行为经济学家乔治.洛温施坦。
